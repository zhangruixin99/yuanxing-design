# 标准肿瘤登记平台功能清单矩阵

日期：2026-06-05

本清单用于配合《标准肿瘤登记平台设计规格》进行需求确认、交付拆分和验收。平台为单省独立部署、一个统一平台，通过行政区划、机构、角色、菜单、操作和数据状态控制不同用户可见功能与可操作范围。

| 一级模块 | 功能 | 功能描述 | 主要交互项 | 验证规则/业务校验 |
|---|---|---|---|---|
| 工作台 | 综合首页 | 按用户角色展示全省/辖区/机构的登记概况、待办任务、质控概览、上传状态和公告资讯。 | 指标卡片、待办列表、地图/区划统计、快捷入口、公告列表、刷新、筛选时间范围。 | 用户只能看到授权区划和机构数据；指标口径与正式库统计口径一致；待办数量需与任务列表一致。 |
| 工作台 | 待办中心 | 汇总审核、退回修正、纠错工单、重卡处理、随访计划、死亡更新、上传失败等任务。 | 任务类型筛选、状态筛选、批量处理入口、跳转详情、催办、导出。 | 待办按角色、菜单权限、数据范围过滤；已处理任务不得重复处理；超时任务按流程配置标记。 |
| 工作台 | 资讯公告 | 发布和展示通知、培训材料、政策文件、系统公告等信息。 | 公告列表、详情查看、附件下载、按类别筛选、已读标记。 | 只能查看本级及下级授权范围公告；附件下载留痕；已撤销公告不展示。 |
| 报告卡管理 | 报告卡登记 | 新增肿瘤发病报告卡，填写发病基本信息、肿瘤报告信息和随访报告信息。 | 新增、暂存、提交、重置、身份证自动解析、地址树选择、字典检索、ICD 自动生成、重卡提示。 | 必填项校验；18 位身份证格式校验；身份证与性别、出生日期一致性校验；户籍/常住地址至少精确到配置要求层级；ICD-10/ICD-O-3 根据部位、亚部位、行为、分级、病理类型生成；错误阻断提交，警告允许提交并留痕。 |
| 报告卡管理 | 报告卡维护 | 对授权范围内报告卡进行查询、查看、编辑和逻辑删除。 | 查询、重置、详情、编辑、删除、按行政区划/校验状态/报卡类型/日期/登记编号/姓名/身份证筛选。 | 仅允许编辑权限范围内且状态允许编辑的记录；删除为逻辑删除并进入作废卡管理；编辑需重新触发校验和审计。 |
| 报告卡管理 | 报告卡查询 | 面向统计、审核、管理用户查询报告卡并导出结果。 | 条件查询、详情查看、导出 Excel/CSV、克隆入口、列设置。 | 导出受权限控制；敏感字段按角色脱敏；克隆仅限登记中心等授权角色；查询结果不得越权。 |
| 报告卡管理 | 报告卡克隆 | 登记中心用户基于辖区内非当前用户上报的正式数据创建调整/合并用新卡。 | 选择原卡、克隆、编辑、提交、取消。 | 仅授权用户可克隆；原卡与克隆卡需保留来源关系；克隆卡提交后走校验、审核、查重流程。 |
| 报告卡管理 | 作废卡管理 | 管理逻辑删除后的报告卡，支持查询、恢复和物理删除。 | 查询、详情、导出、还原、物理删除、批量选择。 | 还原需恢复到删除前状态；物理删除为高风险动作，需单独授权、二次确认和审计；已物理删除不可恢复。 |
| 随访与死亡管理 | 随访信息管理 | 维护存活患者随访记录，展示历史随访并新增最新随访。 | 查询、详情、随访列表、添加随访、修改最新随访、删除最新随访、批量随访上传。 | 随访记录按时间倒序；仅最新随访可修改/删除；至少保留一条有效随访记录；最后接触日期和状态需成对填写。 |
| 随访与死亡管理 | 批量随访 | 通过模板批量更新患者随访信息。 | 模板下载、文件上传、校验结果、错误导出、批量入库、逐条处理。 | 文件格式和字段校验；患者/报告卡匹配校验；死亡状态需补充死亡日期、死因、死亡地点；错误记录不得批量入库。 |
| 随访与死亡管理 | 随访计划 | 按规则生成随访任务并分配给机构或人员。 | 计划生成、任务列表、分派、提醒、完成登记、逾期筛选。 | 计划生成规则按癌种、诊断日期、最后接触状态配置；死亡、作废、归档记录不生成常规随访任务；逾期状态自动计算。 |
| 随访与死亡管理 | 死亡信息管理 | 管理已死亡报告卡及死亡随访记录。 | 查询、详情、随访记录查看、死亡回退、导出。 | 仅死亡状态记录显示；死亡回退仅可删除最新且唯一有效的死亡随访记录；回退后记录回到随访信息管理。 |
| 随访与死亡管理 | 死因补录 | 对死因数据匹配到已有发病记录但尚未更新的病例进行补录。 | 待补录列表、匹配详情、补录表单、提交、驳回、转协查。 | 死因记录必须匹配患者或报告卡；补录死亡信息需满足死亡日期、根本死因、死亡地点等必填规则；补录后记录状态更新并留痕。 |
| 审核与质控 | 审核工作台 | 多级审核人员集中处理待审报告卡、随访、死亡、导入和协查数据。 | 待审列表、详情、确认、退回、驳回、转协查、批量审核、审核意见。 | 审核节点由流程配置决定；无权限节点不可处理；退回必须填写原因；驳回记录不进入正式统计口径；审核结果全程留痕。 |
| 审核与质控 | 可配置审核流程 | 按省份、区划、机构、来源、报卡类型配置审核节点和动作。 | 流程设计、节点配置、条件配置、发布、停用、版本查看。 | 发布前需校验流程有开始和结束节点；节点角色不能为空；已发布流程修改需生成新版本；在途数据按原流程或配置迁移策略处理。 |
| 审核与质控 | 自动质控规则 | 对录入、导入、接口上传数据执行必填、格式、逻辑、编码和自定义校验。 | 规则列表、新增规则、编辑规则、启停、测试规则、发布版本。 | 规则表达式必须可解析；规则结果只能为错误、警告、通过；错误阻断保存或入库；规则变更需记录版本和操作人。 |
| 审核与质控 | 纠错工单 | 对异常数据发起修正任务，跟踪修正、复核和关闭。 | 发起工单、指派、修正、提交复核、关闭、退回、工单日志。 | 工单必须关联业务记录；关闭前需完成复核；工单处理人需具备数据范围权限；所有修正保留前后值。 |
| 审核与质控 | 质量评估报告 | 生成完整性、准确性、一致性、漏报率、错误率、及时率等质量报告。 | 条件筛选、指标查看、图表、明细下钻、导出 PDF/Excel/Word。 | 指标计算口径需固定版本；只能统计授权范围数据；报告导出需记录操作日志。 |
| 重卡与主档案 | 肿瘤登记报告查重 | 对报告卡进行自动或手动查重，识别重复报告和疑似多原发。 | 查重分析、分组展示、展开详情、对比合并、快速合并、标记多原发。 | 查重范围限有效唯一卡；可疑多原发需提示并阻断快速合并；合并后保留来源卡关系；重复处理不可造成数据丢失。 |
| 重卡与主档案 | 发病人口信息查重 | 对患者/发病人口信息进行查重和合并。 | 查重依据选择、查询、分组展示、对比合并、信息合并。 | 查重依据支持身份证、姓名电话、联系人电话、姓名性别出生日期地址等组合；人口合并需同步空间地址、报告卡、随访、死亡记录关系。 |
| 重卡与主档案 | 对比合并 | 多条疑似重复记录并列展示，人工选择保留字段并执行合并。 | 字段对比、默认字段选择、手工选择、重置、合并提交、取消。 | 默认值按合并字段优先级生成；必填字段合并后不能为空；合并前需二次确认；合并日志保留来源和目标。 |
| 重卡与主档案 | 快速合并 | 对无明显冲突的重复记录按规则自动合并。 | 勾选记录、快速合并、结果提示。 | 同组记录才可快速合并；存在可疑多原发或关键冲突时阻断；合并结果需可追溯。 |
| 重卡与主档案 | 多原发管理 | 标记、查看和还原多原发关系。 | 标记多原发、关系查看、快速还原、详情展开。 | 多原发判断需记录依据；还原需恢复标记前状态；多原发记录不按重复卡合并。 |
| 重卡与主档案 | 自动查重参数 | 配置查重字段组合和查重条件。 | 字段组合新增/编辑/删除、条件新增/编辑/删除、与/或逻辑、启停。 | 查重条件必须引用已启用字段组合；字段组合不能为空；删除已引用参数需先停用或解除引用。 |
| 重卡与主档案 | 合并字段参数 | 配置合并时各字段默认取值优先级。 | 字段列表、编辑优先级、启停、说明。 | 优先级策略需在允许范围内；停用字段不得参与自动合并；规则变更需版本留痕。 |
| 基础数据管理 | 发病人口信息管理 | 维护患者基础信息和人口归属。 | 查询、新增、编辑、删除、详情、批量转换区划、过滤行政区划。 | 身份信息校验；人口归属调整需授权；批量转换需记录原区划和目标区划；常住地址联动调整需显式勾选。 |
| 基础数据管理 | 空间地址信息管理 | 维护患者户籍、常住、经纬度等地址历史。 | 查询、地址历史列表、添加、修改最新地址、删除最新地址。 | 地址历史按时间倒序；仅最新地址可修改；地址区划编码必须有效；修改需写入空间地址日志。 |
| 基础数据管理 | 标准人口 | 维护中国标准人口等标准分母数据。 | 年度选择、年龄组录入、添加、编辑、查看。 | 年度不能为空；年龄组人口数为非负整数；同一标准类型和年度不得重复。 |
| 基础数据管理 | 区域人口 | 维护各区划年度、性别、年龄组人口数。 | 添加、编辑、删除、查看、年度筛选、区划筛选。 | 同一区划、年度、性别只允许一组年龄人口数据；县区级默认可编辑，其他层级默认只读；人口数为非负整数。 |
| 基础数据管理 | 死亡人口 | 维护各区划年度、性别、年龄组死亡人口数。 | 添加、编辑、删除、查看、年度筛选、区划筛选。 | 同一区划、年度、性别不得重复；县区级默认可编辑；死亡人口数为非负整数且不应超过同口径区域人口。 |
| 数据采集与交换 | 接口说明与模板中心 | 提供必填说明、上传说明、接口说明和模板下载。 | 模板下载、接口文档查看、字段说明、版本选择。 | 模板版本需与当前字段配置匹配；停用模板不可下载；模板变更需保留历史版本。 |
| 数据采集与交换 | 批量导入 | 支持 Excel、CSV 等文件批量导入报告卡、随访、死因、协查和人口数据。 | 添加文件、上传、校验、批量导入、逐条处理、删除、错误导出。 | 文件类型和大小校验；导入类型合法；必填、格式、字典、逻辑规则校验；仅校验通过记录可批量入库；错误提示需可导出。 |
| 数据采集与交换 | HIS/LIS/PACS/病理接口 | 从医院业务系统自动或半自动采集肿瘤相关诊疗、检查、病理数据。 | 接口任务、手动拉取、自动同步、日志查看、失败重试、字段映射配置。 | 接口认证通过；报文格式合法；字段映射完整；重复数据按来源业务号和主索引规则处理；失败任务可重传。 |
| 数据采集与交换 | 死因上传管理 | 接收死因监测数据并进入匹配、补录和死亡更新流程。 | 文件/接口上传、临时库、匹配结果、逐条处理、批量入库、错误导出。 | 死因类型合法；死亡日期、根本死因等必填；匹配成功生成更新待办；匹配失败进入漏单补录或协查。 |
| 数据采集与交换 | 信息协查管理 | 接收或发起协查数据，支持暂存和入库提交。 | 协查上传、暂存提交、入库提交、逐条处理、转办、反馈。 | 暂存不进入正式库；入库提交需通过报告卡同等校验；协查数据需保留来源和处理状态。 |
| 数据采集与交换 | 外部平台交换 | 与国家癌症中心、省全民健康平台等进行接口或文件交换。 | 交换任务、上传/下载、状态跟踪、失败重传、模拟服务、日志、错误回写。 | 外部接口未提供时按标准适配器和模拟服务验收；任务必须记录请求、响应、错误和重传次数；导出文件需符合目标模板。 |
| 数据采集与交换 | 国家平台数据回流导入 | 支持国家癌症中心导出数据导入本系统并处理重复。 | 文件上传、字段映射、重复识别、预览、入库、错误导出。 | 字段映射完整；重复数据进入查重治理；不得直接覆盖正式数据；导入后保留来源批次。 |
| 统计分析 | 年龄组发病数分析 | 按年龄组、部位、性别、区划、时间统计发病数量。 | 条件筛选、图表/表格切换、导出 Excel。 | 统计范围限正式有效数据；年龄按配置分组；日期类型和区间合法。 |
| 统计分析 | 年龄组发病率分析 | 基于区域人口或标准人口计算年龄组发病率。 | 条件筛选、人口口径选择、导出、下钻。 | 分母人口数据必须存在；发病率计算公式固定并可追溯；分母为 0 时提示不可计算。 |
| 统计分析 | 年龄组死亡数分析 | 按年龄组、部位、性别、区划、时间统计死亡数量。 | 条件筛选、图表/表格切换、导出。 | 仅统计死亡状态或有效死亡记录；死亡日期区间合法；重复死亡记录需去重。 |
| 统计分析 | 年龄组死亡率分析 | 基于死亡人口或区域人口计算年龄组死亡率。 | 条件筛选、人口口径选择、导出、下钻。 | 分母数据必须存在；死亡率公式固定；分母为 0 时提示不可计算。 |
| 统计分析 | 综合部位数据分析 | 按癌种/部位综合展示发病、死亡及趋势。 | 区划、截止日期、报卡范围、时间统计类型、性别筛选、导出。 | 部位分组按 ICD/配置字典执行；统计口径与报表一致；数据范围不得越权。 |
| 统计分析 | 月报表 | 按区划、单位、用户统计月度报告数量。 | 按区划/单位/用户切换、年份、日期类型、报卡范围、导出。 | 月份按日期类型计算；作废、驳回记录不进入正式口径；导出结果与页面一致。 |
| 统计分析 | 报告完整性 | 按区划、单位、用户统计报告完整性情况。 | 年份、日期类型、报卡范围、明细下钻、导出。 | 完整性规则按质控配置计算；缺失项需可下钻查看；统计范围限授权数据。 |
| 统计分析 | 交叉报表 | 用户自定义纵横维度生成交叉统计。 | 日期范围、ICD 分组、年龄分组、纵向项目、横向项目、导出。 | 纵向和横向项目不可相同或冲突；维度值需来自有效字典；大数据量查询需限制或异步导出。 |
| 统计分析 | 多维查询与导出 | 按地区、时间、年龄、性别、肿瘤部位等多维筛选并导出。 | 筛选器、列配置、保存查询条件、导出 Excel/PDF/Word。 | 敏感字段导出按角色脱敏；批量导出需高风险授权；查询条件需合法且可复现。 |
| 结构化年报 | 数据自动汇总 | 按年度、区划、机构层级自动提取年报所需病例、人口和死亡数据。 | 年度选择、区划选择、机构层级、生成汇总、查看明细。 | 仅使用正式有效数据；人口分母完整性检查；缺失分母或关键数据时生成问题清单。 |
| 结构化年报 | 指标预计算 | 自动计算发病率、死亡率、年度变化趋势等核心指标。 | 指标列表、重新计算、计算日志、异常提示。 | 指标公式固定版本；分母为 0 或缺失时提示；计算结果保留批次和时间戳。 |
| 结构化年报 | 年报模板生成 | 按固定章节生成结构化年报初稿。 | 模板选择、章节预览、图表插入、文字编辑、保存草稿。 | 章节模板必须完整；指标占位符必须能解析；人工编辑保留版本。 |
